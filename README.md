Enterprise-Grade Named Entity Recognition (NER) Pipeline
An end-to-end NER project that compares a classical sparse-feature pipeline against a modern embedding and vector-retrieval architecture.

Project Goals
This project is built in two phases:

Phase 1: Bag of Words, n-grams, TF-IDF, and classical classifiers.
Phase 2: Dense embeddings and vector database retrieval.
Initial Scope
The current scaffold includes:

Project structure and configuration files
Dataset ingestion entrypoints
Text preprocessing utilities
CLI entrypoint for first-stage pipeline commands
Planned Milestones
Data ingestion and preprocessing
TF-IDF and n-gram feature extraction
Logistic Regression baseline
Embedding pipeline and Chroma integration
Evaluation, benchmarking, and documentation polish
